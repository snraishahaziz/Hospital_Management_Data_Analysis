# First_SQL_Project
Analysing a hospital management's database to explore number of staff allocated, average
patient admission, feature relationship and more using MySQL.

The project shows understanding of the following:
  - Basic SQL clauses (e.g.: SELECT, FROM, WHERE)
  - Window functions (e.g.: WITH, ROW_NUMBER(), DENSE_RANK())
  - Creating tables and database
  - Importing loading dataset into MySQL and inserting the dataset into
    tables

The project is divided into the following 4 sections, each sections representing one dataset (table) under the database hospital_management:
  - staff: data on staff working at the hospital
  - schedule: data on staff's presence and absence
  - service: data on the service's provided by the hospital including staff
             morale and patient's satisfaction
  - patients: data on patients admitted to the hospital

Each queries serve different purposes, hence repeated queries may exist but with different purpose as indicated by the comments made before each query. Further report on the data analysis can be found in the file "Hospital_Management (SQL Project)".

The dataset is artificially generated and taken from https://www.kaggle.com/datasets/jaderz/hospital-beds-management.
